## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

## 功能
- 阿里oss   静态托管网站
- 阿里cdn   内容分发节省流量
- 阿里流水线 代码监听推送部署
- 阿里微队列
